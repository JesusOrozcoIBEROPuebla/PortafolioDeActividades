Attiny85 Electrónica

- Resumen -

En esta practica se creo un Wearable estilo de reloj que utiliza 3 sensores; para medir la oxigenacion de una persona, un sensor de impacto para alertar que el sujeto sufrio una caida y otro sensor de presion barometrica para conocer la altura a la que se encuentra y todo estos valores se muestra en una pantalla OLED

- Introducción -

Se investigo los componentes y sensores que probrablemente se ocuparian pero tomamos la decision de esos 3 sensores: Oxigenacion, impacto y barometrica

- Materiales -

Los materiales ocupados en estas practicas son los siguientes: Arduino UNO, Software Arduino, archivos descargables de cada practica, cables macho-macho, push bottom, pantalla OLED, Modulo KY-031, Sensort de impacto, Sensor MBP 180, y sensor Max30102

- Desarrollo -

Como primer paso abrimos el código arduino, antes de compilarlo y/o iniciar el código debemos especificar el tipo de arduino que vamos a utilizar, en nuestro caso Arduino UNO. Una vez seleccionado damos click en "ok". Dentro de los códigos tendremos varias funciones establecidas, se deberá seguir el código para poder realizar las conexiones a los pines de salida.
